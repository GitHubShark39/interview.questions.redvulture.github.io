Prerequisites
Before you begin, you will need an Ubuntu 16.04 server with at least 1G of RAM. 
Complete the following guides to set up a non-root user, install and configure 
Concourse, install Nginx, obtain a TLS/SSL certificate, and set up a secure 
reverse proxy to the Concourse web UI. You will need a domain name pointed at 
your Concourse server to properly secure it:
